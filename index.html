Quick Start

One command to
see your whole codebase.

~ npm i -g @depic/cli
added 47 packages in 3s

~/project depic analyze --dot
digraph {
"src/index.ts" -> "src/utils.ts"
"src/utils.ts" -> "node_modules/lodash"
...
}

~/project depic serve --port 3000
Server running at http://localhost:3000
Packages

Pick what you need.

⬡

@depic/core

Core engine. Parse files with SWC, resolve module specifiers, build directed graphs, detect cycles, compute topological order.

npm i @depic/core

⌨

@depic/cli

Terminal-first tool. Analyze, detect cycles, find dependents, generate HTML reports, or start a live server.

npm i @depic/cli

◈

@depic/web

Interactive graph UI. Generate standalone HTML reports or start a local server with live graph browsing and REST API.

npm i @depic/web

⊞

depic-vscode

Full IDE integration. Visualize the dependency graph, check cycles, and find dependents without leaving VS Code.
